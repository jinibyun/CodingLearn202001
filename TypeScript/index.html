Pre-requisite

  • install node.js
  • install npm install -g typescript
  • (ref: https://www.typescriptlang.org )
  • Confirmation
  • 1. node -v
  • 2. npm -v
  • 3. tsc -v
  • 4. Then opne this folder with vs code and Setup environment for learning TypeScript (http://www.mithunvp.com/learning-typescript-with-visual-studio-code/)
  • Create ts file(s) and tsc to compile all ts file
  • For browser sync, type live-server in command prompt (Ctrl+Shift+P to type "live-server" to start and stop)

Please check it out console window (F12) on browser to see the result
